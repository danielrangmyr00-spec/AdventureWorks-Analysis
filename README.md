AdventureWorks – Försäljningsanalys

Denna notebook innehåller en försäljningsanalys av AdventureWorks-databasen.
Analysen är genomförd med SQL Server för datahämtning och Python för bearbetning och visualisering.

Innehåll
- Produktanalys
- Försäljning över tid
- Regional försäljning
- Kundanalys

Resultaten presenteras i form av visualiseringar och affärsinsikter.
